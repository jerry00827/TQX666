✨ 乘法分配律 ✨

—— 拆括号，分着乘，合起来，简便算
🧙 魔法公式
(a + b) × c = a × c + b × c
a × (b + c) = a × b + a × c

一个数乘两个数的和，等于这个数分别乘这两个数，再相加。

📖 反过来也成立：两个乘积的和，如果有相同的因数，可以合并成乘法。

🎈 小口诀：

✔️ 括号外面乘进来，分别乘完再相加。
✔️ 合并时找相同数，提取出来乘括号。
✔️ 巧用分配律，计算快又准！

(3+4)×5 = 3×5+4×5
6×(2+7) = 6×2+6×7
8×9+8×1 = 8×(9+1)

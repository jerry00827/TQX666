📏 数轴上的数

—— 正数、0、负数的位置
← -3 -2 -1 0 1 2 3 →

✨ 数轴的三要素：原点、正方向、单位长度。

📖 数轴是一条直线，原点（0）把数轴分成左右两边：

✔️ 0的右边是正数，越往右越大。
✔️ 0的左边是负数，越往左越小。
✔️ 正数 > 0 > 负数。

🎯 例如：-3 在 0 的左边，2 在 0 的右边。

🎈 小口诀：

✔️ 画直线，定原点，向右箭头正方向。
✔️ 单位长度要相等，正数右边负数左。
✔️ 数轴好比温度计，上正下负要分清。

-2 在 0 的左边
+3 在 0 的右边
0 是分界点

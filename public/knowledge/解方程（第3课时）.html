📐 解方程

—— 第3课时：两步方程 ax±b=c, a(x±b)=c
2x + 3 = 13
解：2x + 3 - 3 = 13 - 3
2x = 10
x = 5
3(x + 2) = 18
解：3(x+2) ÷ 3 = 18 ÷ 3
x + 2 = 6
x = 4

✨ 解两步方程的方法：

先利用等式的性质将方程化简为一步方程，再求解。

📖 步骤：
✔️ 形如 ax ± b = c：先加减消去b，再乘除解出x。
✔️ 形如 a(x ± b) = c：先两边同时除以a，再加减解出x。
✔️ 最后检验。

🎯 例如：
4x - 5 = 11 → 4x = 16 → x = 4。
2(x - 3) = 10 → x - 3 = 5 → x = 8。

🎈 小口诀：

✔️ 有加减先加减，有括号先去括号。
✔️ 两边同除系数，x值轻松得。
✔️ 第3课时两步走，细心计算全做对。

3x+2=11 → x=3
5(x-1)=20 → x=5
4x-3=13 → x=4
